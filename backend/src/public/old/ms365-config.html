menu_book Setup Instructions
1

Register an App in Azure AD

Go to Azure Portal → App registrations and create a new registration.

Name: Agent5
Supported account types: Single tenant
2

Create a Client Secret

In your app registration, go to "Certificates & secrets" → "New client secret". Copy the Value immediately!

3

Configure API Permissions

Add these Microsoft Graph Application permissions:

Mail.Read, Mail.ReadWrite, Mail.Send
Calendars.Read, Calendars.ReadWrite
User.Read.All
Files.Read.All, Files.ReadWrite.All
Team.ReadBasic.All, Channel.ReadBasic.All
warning Click "Grant admin consent" after adding permissions!
4

Add Environment Variables

Add these to your server's .env file:

MS_TENANT_ID=your-tenant-id
MS_CLIENT_ID=your-client-id
MS_CLIENT_SECRET=your-client-secret
MS_USER_ID=nima@acropaq.com

Find Tenant ID and Client ID on the app's "Overview" page in Azure.
