menu_book Setup Instructions
1

Get API Credentials from Bol.com

Log in to your Bol.com Seller Dashboard. Go to Settings → API Settings and create new API credentials.

2

Copy Client ID and Secret

After creating the credentials, you'll receive a Client ID and Client Secret. Copy these values immediately - the secret is only shown once!

warning Store your Client Secret securely - it cannot be retrieved later!
3

Add Environment Variables

Add these to your server's .env file:

BOL_CLIENT_ID=your-client-id
BOL_CLIENT_SECRET=your-client-secret
4

Restart the Server

After adding the environment variables, restart the Agent5 server to apply the changes.

pm2 restart agent5
