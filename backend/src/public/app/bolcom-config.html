menu_book Setup Instructions
1

Get Retailer API Credentials

Log in to your Bol.com Seller Dashboard. Go to Settings → API Settings → Client credentials voor de Retailer API and create new credentials.

2

Get Advertising API Credentials (Separate!)

In the same API Settings page, go to Client credentials voor de Advertising API and create separate credentials for advertising.

warning The Advertising API requires different credentials than the Retailer API!
3

Copy All Credentials

Copy both sets of credentials immediately - secrets are only shown once!

warning Store your secrets securely - they cannot be retrieved later!
4

Add Environment Variables

Add these to your server's .env file:

# Retailer API (orders, shipments, returns, offers)
BOL_CLIENT_ID=your-retailer-client-id
BOL_CLIENT_SECRET=your-retailer-client-secret

# Advertising API (campaigns, costs, performance)
BOL_ADVERTISER_ID=your-advertiser-id
BOL_ADVERTISER_SECRET=your-advertiser-secret
5

Restart the Server

After adding the environment variables, restart the Agent5 server to apply the changes.

pm2 restart agent5
